# Tracy Prompt Agent

Deploy-ready FastAPI agent using OpenRouter + Render.

## Features
✅ Modular
✅ SQLite local memory
✅ Logs responses
✅ Render/Docker ready

## Environment Variables
- OPENAI_API_KEY: (your OpenRouter key)
- MODEL: openchat/openchat-3.5 (or other)

## Endpoints
- POST /prompt-agent

## Deploy
1. Push to GitHub
2. Link to Render
3. Add your environment variables
4. Deploy
